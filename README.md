# Research-portfolio

Personal research portfolio — independent work in Flutter, ECC, and decentralised systems.

## What this is

- A living archive of projects built from first principles
- Documentation of ongoing cryptographic research (elliptic curves, SafeCurves compliance)
- A contact point for collaboration in applied cryptography and peer-to-peer systems

## Projects featured

- **Chatme** – Nostr-based, serverless chat with WebRTC voice
- **Curve256189** – Original ECC library in Dart (SafeCurves complete)
- **Curve33617** – Montgomery curve at ~168-bit security level
